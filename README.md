# SIFIX Documentation

Official documentation for SIFIX - AI-Powered Wallet Security for Web3.

## Development

```bash
# Start dev server
npx @docmd/core dev

# Build for production
npx @docmd/core build

# Preview production build
npx @docmd/core preview
```
